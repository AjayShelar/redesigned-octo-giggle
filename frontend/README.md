# Frontend

React + Vite frontend for the workflow UI.

## Tech Stack (MVP)

- Node.js 20+ — local runtime and tooling
- React 19 — UI framework for core views
- Vite 7 — fast dev server and build tooling
- TypeScript — type safety across UI logic
- Tailwind CSS — utility-first styling and design tokens

## Local Development

From the repo root:

```bash
cd frontend
npm install
npm run dev
```
